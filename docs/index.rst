Welcome to Nectar's Water Quality Sensor documentation!
=======================================================

The Nectar Water Quality Sensor platform provides tools for testing
the quality of water for botanical systems. We implement open source 
software such as TI-RTOS, and InfluxDB for a reliable and cutting edge
approach.
